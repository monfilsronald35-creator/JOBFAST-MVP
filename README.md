# 🚀 JOBFAST

MVP Platform: Construction + Services + Business Directory + GPS Jobs System

---

## 📌 OVERVIEW

JOBFAST se yon platform ki konekte:

- 🏗️ Construction workers (boss, worker, apprentice, engineer)
- 🧑‍🔧 Services on demand (plumber, doctor, taxi, cleaner, etc.)
- 🏢 Business directory (companies & organizations)
- 📍 GPS-based matching system (nearby jobs & workers)

👉 Objektif: konekte moun ak travay yo rapid selon kote yo ye.

---

## 🌍 LANGUAGE SYSTEM (IMPORTANT)

Platform nan sipòte 3 lang:

- 🇭🇹 Kreyòl (ht)
- 🇫🇷 Français (fr)
- 🇪🇸 Español (es)

👉 Tout UI + notifications adapte otomatik selon language user.
